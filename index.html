Jump On Brick

"Jump on Bricks" is an engaging web-based game developed using HTML, CSS, and JavaScript, offering a fun and challenging experience. The player controls a character that jumps from one brick to another, aiming to avoid falling while collecting points. JavaScript handles the game logic and real-time physics, ensuring smooth interactions and responsive gameplay. This project combines creativity and functionality to deliver an addictive, fast-paced gaming experience.

3D Cube Design

The 3D cube design is an interactive visual element created using HTML and CSS, adding depth and creativity to web pages. By leveraging CSS transforms and animations, each face of the cube is styled and rotated to create a dynamic, three-dimensional effect. The cube can spin, flip, or hover based on user interactions, enhancing user engagement. Its smooth transitions and realistic perspective offer a modern and visually striking design. This project highlights the power of CSS in crafting interactive 3D elements without the need for external libraries or JavaScript.
